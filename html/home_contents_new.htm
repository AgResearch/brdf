Welcome to the AgResearch Expression and Sequence Database

News

October 2008

  • Keyword Search Update

    The presentation of search results has been improved, with the addition of an intermediate screen that categorises matching sequences - e.g. distinguishing matches to AgResearch Cattle, Sheep, Deer, Clover, Ryegrass and other species , Affymetrix probe seqs, public seqs etc. From these links you can click through to lists of the hits and through to individual sequence pages.

    The search engine has been updated to improve keyword searches. Searches now list AgResearch seqs that have hits to keyword-matching NCBI / DFCI / Interpro etc sequences directly. Previously you had to click through to the original AgResearch query from the matching hit via the information map. The engine searches sequence names, descriptions and blast hits - where the match is to a blast hit, the search summary indicates the matching hit sequence.

  • Updates to Sequence Page Web Interface
    The standard web page for a sequence now includes a menu item "View" with two items "as FASTA" and "as Genbank".

    For most users, the "as Genbank" item will automatically open the sequence in Vector NTI. The Genbank file includes any features annotated for the sequence.

  • Linking to sequences in the database
    You may now link directly to Genbank and FASTA formatted records, as well as the default html view. Examples :

    /cgi-bin/fetch.py?obid=CS37.231105CS3700262300001&context=default&target=ob # default HTML

    /cgi-bin/fetch.py?obid=CS37.231105CS3700262300001&context=genbank&target=ob # Genbank

    /cgi-bin/fetch.py?obid=CS37.231105CS3700262300001&context=fasta&target=ob # FASTA
