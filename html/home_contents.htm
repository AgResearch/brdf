25 September 2015

    Reviewing Illumina sample-sheets and making custom GBS keyfiles
  • You can review sample-sheets associated with the AgResearch Hiseq using the Quick Reports link on the left.
  • You can get a listing of GBS keyfiles using the Quick Reports link on the left. To make a custom keyfile :
    1. click on the GBS Keyfiles Extract link under Quick Reports to run the report
    2. on the resulting page, right click and choose "Export to Excel"
    3. wait patiently
    4. you can now use Excel filtering to choose the rows you want
    5. save the filtered result as a tab-delimited text file - this is your custom keyfile

4 July 2015

    Browsing Otago Illumina HighlandersSeq Runs, Samplesheets and GBS Keyfiles
  • Each run has a page with a URL like this http://agbrdf.agresearch.co.nz/cgi-bin/fetch.py?obid=150626_D00390_0231_BC6JT3ANXX&context=default (- the page for run 150626_D00390_0231_BC6JT3ANXX )

  • Each Sample Plate has a page with a URL like this http://agbrdf.agresearch.co.nz/cgi-bin/fetch.py?obid=SQ2525&context=default ( - the page for sample plate SQ2525 )

  • You can also search for sample plates by entering all or part of the plate name in the top search field at http://agbrdf.agresearch.co.nz , and choosing "samples" as the type of records to search for

  • From the run page you can click through to the sample sheet associated with that run, or to the individual sample plate pages.

  • From a sample plate page, you can click through to view the keyfile associated with that sample plate, and also back through to the hiseq runs that are associated with that plate.

  • The database is mainly used to manage keyfiles (e.g. fastq locations), and make it easier to do basic reporting and Q/C checks on runs x sample plates x samples x keyfiles x fastq locations. Let us know if you need any reports or batch updates set up and run.
